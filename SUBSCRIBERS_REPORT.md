# Paid Subscribers Report

## Dodo Payment Renewal Verification

**Yes — subscriptions continue automatically when users don't cancel.**

The Dodo webhook (`/api/dodo/webhook`) handles recurring billing as follows:

| Event | What Happens |
|-------|--------------|
| **subscription.renewed** | Dodo charges the customer → webhook receives event → Firestore updated with new `startDate`/`endDate` → credits reset to full plan amount |
| **payment.succeeded** | Safety net: if payment event arrives before subscription event, subscription is activated/updated |
| **subscription.cancelled** / **subscription.expired** | User downgraded to free; access ends at period end |

**Flow:** Dodo charges the card each billing cycle. If the charge succeeds, Dodo sends `subscription.renewed` (or `payment.succeeded`). The webhook updates the user's Firestore doc with the new period and resets credits. If the user cancels, Dodo sends `subscription.cancelled` and we downgrade them.

**Conclusion:** As long as the user does not cancel and Dodo successfully charges their card, their subscription will continue and credits will reset each month.
